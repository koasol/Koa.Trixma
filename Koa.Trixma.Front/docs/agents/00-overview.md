# AI Assistant Overview

This directory contains high-signal information specifically for AI agents (like Junie). These documents are designed to be succinct and easily parsed.

## Core Documents

1. **[Context Map](./10-context-map.md)**: Where things are and what they do.
2. **[Goals & Non-Goals](./20-goals.md)**: What we are trying to achieve and what is out of scope.
3. **[Requirements & Constraints](./30-requirements.md)**: Tooling, env vars, and technical limitations.
4. **[Edit Policy](./50-edit-policy.md)**: Safety rules for modifying the codebase.

## How to use this context
- **Check the Edit Policy** before modifying any non-UI code.
- **Reference the Requirements** if build or deployment fails.
- **Consult the Context Map** when searching for the source of a specific feature or behavior.
